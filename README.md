1. PlatformIO
2. Wokwi Simulator

Sau khi cài đặt xong 2 extension thì folder .pio và .vscode sẽ tự động được tạo ra
Nếu ae mới vừa cài extension, thời gian đợi máy tự cài đặt folder .pio .vscode và cập nhật thư viện trong platformio.ini có thể hơi lâu (vài phút)

# CHẠY CHƯƠNG TRÌNH
## Bước 1. Build chương trình
Có 2 cách để build chương trình:
Cách 1: Vào diagram.json, ở góc trên bên phải màn hình, chọn Build
Cách 2: Vào diagram.json, Ctrl + Shift + P --> Nhập "PlatformIO: Build"

## Bước 2. Chạy chương trình
Vào diagram.json, ở góc trên bên phải màn hình, chọn Run Code
